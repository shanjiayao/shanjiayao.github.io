摘要

提出了一种新型的点云编码方式，叫做PointPillars, pillar就是点云按照竖直柱子编码的方式，可以理解成高度方向不限制的体素化编码方式

使用pointnet学习每个pillar的特征

将pillar编码后，可以接很多2D的卷积网络进行检测任务

论文中给出了这种编码方式的好处：

  1. 学习每个pillar的特征，可以更加完整地学习到点云的特征信息
  2. pillar的操作比体素要更加简洁、高效，因为是将学习到的pillar特征压缩到了二维图像上，进而使用二维卷积，更加的快速
  3. pillar的编码方式避免了体素化过程中对高度方向上的手动编码
  4. pillar的编码方式还有一点好处就是，可以适应不同线数的激光雷达

网络

  • 输入输出

    整帧点云 <=> 3D预测框

  • 网络结构

    1. Pillar Feature

      N × 3 ⇒ P × N × D ⇒ P × N × C ⇒ P × C ⇒ H × W × C

      P 是 non-empty pillar number， 具体使用 12000
      N 是一个pillar内有points的数目， 具体本文使用100
      D 是 channel number

      • 点云栅格化，对俯视图做离散化，也就是栅格化，得到 H×W个栅格，每个栅格算上竖直高度的信息，就是一个pillar，这里只取P个含有点的pillar，所以 P < H × W

      • 信息编码，对P个pillars操作，每个有效的pillar采样出 N 个点，每个点用九个参数 D 来表示，那么每个点得到 D × N的编码特征表示，最后总体就是 P × N × D 维度

        关于D：维度为9，参数分别是 {x,y,z,r,xc,yc,zc,xp,yp}

        x y z是当前点的坐标

        r 代表 reflectance反射率

        Xc Yc Zc是当前点与所有点计算出来均值点（pillar中心点）的距离

        Xp Yp是当前点对于整个网格中心的偏移量，这是在水平维度上计算的，所以只有两个参数

      • 特征学习，用PointNet学习特征

        每个pillar的特征可以看做 N × D，要做的就是使用PointNet，对N个点进行特征升维

        从 D 维中学习 C 个通道的特征，组成 P × N × C 的张量

        然后在N个点的维度上做最大池化，得到 P × C

        再将 P 个pillars 的维度 reshape 成 H × W，得到 H × W × C

        这样点云就转变成了 pseudo-image，每个像素点上带有的是经过编码之后的pillar特征，为

    2. 2D Detector

      基于SSD图像检测算法，进行后续的卷积以及目标框的检测回归
